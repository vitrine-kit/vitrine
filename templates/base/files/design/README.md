# /design — экспорт из Claude Design

Положите сюда экспорт дизайна (React/HTML/CSS, токены, скриншоты) и запустите
`vitrine design apply`. ИИ-шаг прочитает эту папку и инструкцию из `CLAUDE.md`,
затем задаст **только значения токенов** в `theme/client.css` (палитра,
типографика, отступы, радиусы, тени).

Папка изолирована от поведения: её содержимое не влияет на логику, данные,
роутинг и a11y — только на облик через токены. Регенерация дизайна безопасна.
