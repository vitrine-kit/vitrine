# @maks417/vitrine (CLI)

Инструментарий Vitrine. Устанавливается глобально: `npm i -g @maks417/vitrine`.

Команды (§9 спеки): `kit update`/`kit status`/`self-update`; в репозитории клиента — `init` (визард), `add`, `remove`, `update`, `diff`, `list`, `design apply`, `doctor`.

Ядро — **примитив установки фичи**, общий для `init` и `add` (гарантия: «добавлено визардом» ≡ «добавлено позже»).

Скелет (M0). Примитив + `init`/`add` — M4.
